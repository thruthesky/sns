# 🚀 한바보 - AI와 함께 만드는 소셜 서비스

> 코딩 없이 AI와 협업하여 실전 웹/앱을 개발하는 스터디 커뮤니티
> 진행중인 프로젝트: [vibers.kr](https://vibers.kr)

## 💡 프로젝트 소개

**한바보**(한국 바이브 보스)는 AI 기반 개발(바이브 코딩)으로 실전 SNS 웹/앱을 만드는 **오픈소스 스터디 커뮤니티**입니다.

**핵심 원칙**: 소스 코드를 직접 타이핑하지 않고 AI에게 요청하여 구현합니다.

### 📌 프로젝트 변경 사항

본 프로젝트는 실전 적용을 위해 **Svelte 5 + Firebase + Flutter** 기반으로 전환되었습니다.
- 이전: React.js + Next.js + Firebase + Supabase + Flutter
- 현재: Svelte 5 + Firebase + Flutter (SvelteKit 미사용)

### 🎯 100% 바이브 코딩 원칙

- **직접 코딩 절대 금지** - 실수로라도 소스 코드를 직접 수정하면 반드시 원상 복구
- **코드 읽기는 가능** - 소스 코드를 읽고 분석하여 AI에게 명확한 개발 방향 제시
- **가독성 중요** - AI에게 "이해하기 쉬운 코드"로 생성하도록 요청하는 것이 핵심

### ✨ 특징

- 💬 **코딩 금지** - AI가 코드 작성, 당신은 설계와 기획에 집중
- 🛠️ **실전 프로젝트** - 인증, 게시판, 채팅, 알림 등 실제 서비스 기능 구현
- 🤝 **협업 경험** - Git PR, 코드 리뷰, 배포까지 완전한 개발 프로세스
- 🌐 **도구 자유** - 어떤 AI든 사용 가능 (Claude, ChatGPT, Cursor 등)

## 🛠️ 기술 스택

- **웹**: Svelte 5 + Firebase
- **앱**: Flutter + Firebase
- **협업**: Git, GitHub

## 🚀 빠른 시작

```bash
# 1. 저장소 클론
git clone https://github.com/thruthesky/sns.git
cd sns/web

# 2. 의존성 설치
npm install

# 3. Firebase 설정
cp .env.example .env
# .env 파일에 본인의 Firebase 설정 추가

# 4. 개발 서버 실행
npm run dev
```
